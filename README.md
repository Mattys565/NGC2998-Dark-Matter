# NGC2998 Dark Matter Modelling

Analyse de la courbe de rotation de la galaxie NGC2998 et modélisation du halo de matière noire.

## Contenu du dépôt

- `NGC2998(5).ipynb` : Notebook principal avec le code et les calculs
- `Courbe_de_rotation_et_modélisation_de_halo_CDM.pdf` : Rapport de stage détaillé
- `.gitignore` : fichiers à ignorer par Git
- Licence MIT

## Installation

Pour exécuter le notebook, il est recommandé d'utiliser un environnement Python avec les bibliothèques suivantes :

```bash
pip install pip install numpy scipy matplotlib pandas jupyter
```
